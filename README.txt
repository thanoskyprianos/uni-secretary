Τρέξτε `make help` για οδηγίες

Η εκτέλεση του προγράμματος γίνεται ως εξής:

    ./main <students-file> <professors-file> <courses-file> [studying-years] [ects-required-to-graduate]

Υπάρχουν έτοιμα αρχεία στον φάκελο Input, οπότε η εκτέλεση μπορεί να γίνει ως

    ./main Input/students.csv Input/professors.csv Input/courses.csv 4 240

Με 4 χρόνια φοίτησης και 240 ECTS για αποφοίτηση.

Τα στοιχεία δημιουργήθηκαν με το εξής εργαλείο: https://www.random-name-generator.com/

Ερωτήματα:

Β1,2) Οι κλάσεις Student και Professor είναι υλοποιημένες στα αρχεία Person/person.* .
Β3  ) Η κλάση Course είναι υλοποιημένη στα αρχεία Course/course.* .
B4  ) * Το ακαδημαϊκό εξάμηνο υλοποιείται από μια κλάση Semester στα αρχεία Semester/semester.*.
      * Η κλάση Secretary περιλαμβάνει ένα vector απο Semester με το πιο πρόσφατο να είναι
        το τωρινό (semesters.back()).
      * Το πρόγραμμα λειτουργεί ως εξής:
        - Για να καταχωρηθούν οι βαθμοί των μαθητών θα πρέπει να μεταβούμε σε επόμενο εξάμηνο
        - Επομένως, καταχωρούνται οι βαθμοί σε ένα map
        - Και στη συνέχεια, δημιουργείται ένα νέο Semester με τα κατάλληλα μαθήματα (που είναι του ίδιου εξαμήνου)
          και γίνεται push_back() στο vector.
    
B5) Μενού:

1) * Προσθήκη καθηγητή:          | Professors -> Add
   * Διαφοροποίηση καθηγητή:     | Professors -> Search -> Edit
   * Διαγραφή καθηγητη:          | Professors -> Search -> Delete
---------------------------------|------------------------------------------- 
2) * Προσθήκη μαθητή:            | Students -> Add
   * Διαφοροποίηση μαθητή:       | Students -> Search -> Edit
   * Διαγραφή μαθητή:            | Students -> Search -> Delete
---------------------------------|------------------------------------------- 
3) * Προσθήκη μαθήματος:         | Courses -> Add
   * Διαφοροποίηση μαθήματος &   | 
     Αλλαγή εξαμήνου (Β3):       | Courses -> Search -> Edit
   * Διαγραφή μαθήματος:         | Courses -> Search -> Delete
---------------------------------|------------------------------------------- 
4) Ορισμός καθηγητών μαθήματος:  | Courses -> Search -> Assign Professors
---------------------------------|------------------------------------------- 
5) Εγγραφή μαθητή σε μάθημα:     | Students -> Search -> Register Courses
---------------------------------|------------------------------------------- 
6) Εμφάνιση και αποθήκευση       | 
   φοιτητών που πέρασαν σε       | 
   ένα μάθημα σε κάποιο εξάμηνο: | Courses -> Search -> Who passed
                                 | 
   (Το αρχέιο θα αποθηκευτεί     | 
    στον φάκελο Grades)          | 
---------------------------------|------------------------------------------- 
7) Στατιστικά καθηγητή για       | 
   τα μαθήματα του:              | Professors -> Search -> Statistics
---------------------------------|------------------------------------------- 
8) Αναλυτική βαθμολογία μαθητή:  | Students -> Search -> Statistics
---------------------------------|------------------------------------------- 
9) Φοιτητές που μπορούν να       | 
   πάρουν πτυχίο:                | Semester -> Who graduate
